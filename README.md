# 🚚 Roadoz Courier Backend API

A high-performance backend service for the **Roadoz Courier Management System**, built using **FastAPI**. This API powers the admin dashboard and mobile/web applications, enabling efficient shipment tracking, user management, and logistics operations.

---

## 📌 Features

- 🚀 Fast and scalable REST API
- 📦 Shipment Management (Create, Update, Track)
- 👥 User & Admin Management
- 🔐 JWT Authentication & Authorization
- 📍 Delivery Status Updates
- 📊 Reports & Analytics APIs
- 🗂️ Structured and modular architecture
- ⚡ High performance with async support

---

## 🛠️ Tech Stack

- **Backend Framework:** FastAPI
- **Language:** Python 3.9+
- **Database:** PostgreSQL / MySQL / SQLite
- **ORM:** SQLAlchemy / Tortoise ORM
- **Authentication:** JWT (OAuth2)
- **Server:** Uvicorn / Gunicorn
- **Validation:** Pydantic
